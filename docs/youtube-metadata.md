## Title

Verify, Audit, and Pretest X Algorithm Claims with Open Feed Recsys Lab

## Description

Open Feed Recsys Lab is a Codex and ClawHub skill set that turns `xai-org/x-algorithm` into commit-pinned reproducibility reports, source-backed claim audits, and bounded synthetic audience research for draft posts.

The reproducibility skill generates:

- `run_report.md` for commit, repo status, components, and warnings
- `artifact_check.md` for Git LFS and Phoenix artifact readiness
- `architecture_map.html` for a shareable feed architecture map
- `manifest.json` for machine-readable follow-up tooling

The claim auditor generates verdicts and share cards for viral claims. The TinyTroupe-inspired research skill compares draft posts with synthetic personas and writes a report, CSV reaction table, JSON ledger, and TinyTroupe experiment plan.

ClawHub install:
```bash
clawhub install open-feed-recsys-lab
clawhub install x-algo-claim-auditor
clawhub install tinytroupe-feed-research-lab
```

This is not an algorithm growth hack. It does not promise reach, ranking, virality, revenue, or live-platform equivalence. It checks what source was inspected, whether Phoenix artifacts are present, where docs or configs disagree, whether claims are supported by public files, and how synthetic personas react to drafts.
