# SEO Launch Plan

## Decision

Publish canonical tutorial content on GitHub Pages and GitHub README first.

Why:

- The product is a developer tool, so GitHub has the strongest intent match.
- GitHub Pages can rank for long-tail queries such as `xai-org x-algorithm run Phoenix locally`.
- The canonical URL can point to the repo and ClawHub install surface.
- Medium can be used later as syndication, but should not be the canonical URL.
- Notion is useful as an internal launch note, but weak as the primary SEO surface.

## Primary Keywords

- xai-org x-algorithm
- X For You algorithm
- Phoenix recommender
- run X algorithm locally
- feed recommendation system reproducibility
- Codex skill recsys
- ClawHub skill recsys
- Git LFS Phoenix artifacts

## SEO Title

Open Feed Recsys Lab: Verify the X For You Algorithm Repo Locally

## Meta Description

Generate commit-pinned reproducibility reports for xai-org/x-algorithm. Check Phoenix artifacts, Git LFS readiness, documentation drift, and architecture maps for open feed recommendation systems.

## Publication Surfaces

- Canonical: GitHub Pages at `https://zack-dev-cm.github.io/open-feed-recsys-lab/`
- Source: GitHub repo
- Install: ClawHub `open-feed-recsys-lab`
- Video discovery: YouTube tutorial
- Syndication later: Medium and Notion import from `docs/tutorial-medium-notion.md`

## Do Not Claim

- Do not claim the public repo fully matches live production.
- Do not claim the tool predicts reach, virality, or ranking.
- Do not claim Phoenix was run unless extracted LFS artifacts are present.
